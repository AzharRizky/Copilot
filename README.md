# Github Copilot

### Keyboard Shortcut:
- CTRL + Enter = List of Solve Code
- ALT + \[ = Move to Next Solve Code
- Tab = Submit Copilot Code

### How To Use
- With Comment<br>
  <code>// Create some function</code><br>
  <code>function </code>(space & wait code load)

- With Function<br>
  <code>function getDate</code>(wait code load)
